Figure generation
=================

These examples show how GeoPrior-v3 turns model outputs into
interpretable scientific figures.

The goal is not only to make publication-ready visuals, but
also to help readers understand what each panel means, what
data are required, and how to interpret spatial patterns,
contrasts, and physically meaningful inconsistencies.

The examples in this section are written as short lessons.
Each one introduces the scientific question, builds or loads
a compact example dataset, renders the figure, and explains
how to read it.
