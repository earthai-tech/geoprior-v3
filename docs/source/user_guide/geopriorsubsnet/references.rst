.. _geoprior_references:

==========
References
==========

This page collects the bibliography used for the GeoPrior documentation and
the accompanying manuscript. It is intentionally *project-focused* (subsidence,
groundwater extraction, InSAR, risk, and physics-informed / ML methods).

How to cite GeoPriorSubsNet
---------------------------

If you use GeoPriorSubsNet (GeoPrior) or results derived from it, please cite:

.. code-block:: bibtex

   @unpublished{kouadio_geopriorsubsnet_nature_2025,
     author  = {Kouadio, Kouao Laurent and Liu, Rong and Jiang, Shiyu and
                Liu, Zhuo and Kouamelan, Serge and Liu, Wenxiang and
                Qing, Zhanhui and Zheng, Zhiwen},
     title   = {Physics-Informed Deep Learning Reveals Divergent Urban Land Subsidence Regimes},
     journal = {Nature Communications},
     note    = {Submitted},
     year    = {2025}
   }
